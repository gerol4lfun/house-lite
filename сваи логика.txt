Как посчитать сваи (когда калькулятор глючит)

Главное правило (запомнить раз и навсегда)

Мы ставим сваи в рядах вдоль длины, а в каждом ряду — несколько свай по ширине.
Итог = (сколько рядов) × (сколько свай в одном ряду).

⸻

Шаг 1. Определи размер(ы) строения
 • Если просто дом/хозблок/бытовка — размер Длина × Ширина (в метрах).
 • Если есть веранды/пристройки — это доп. прямоугольники, которые «прилипают» к дому. Важно: если веранда не на всю длину, она влияет не на все ряды (см. Шаг 4).

Записываем:
 • Длина = L (метров)
 • Ширина = W (метров)

⸻

Шаг 2. Сколько рядов по длине?

Делаем так, чтобы между рядами было не больше 3 метров.
 • До 3 м длина → 2 ряда (по краям)
 • 3–6 м → 3 ряда (края + середина)
 • 6–9 м → 4 ряда (края + 2 внутри)
 • 9–12 м → 5 рядов
 • и т.д. каждые ~3 м добавляем ещё один ряд.

Подсказка:
L=6 → 3 ряда; L=7 → 4 ряда; L=10 → 5 рядов.

⸻

Шаг 3. Сколько свай в одном ряду по ширине?

Берём минимум 2 сваи в ряд. Дальше — просто по одному «столбику» на каждый метр ширины (вверх округляем).
 • W=2 → 2 сваи
 • W=2.1…3 → 3 сваи
 • W=3.1…4 → 4 сваи
 • W=4.1…5 → 5 свай
и т.д.

⸻

Шаг 4. Что делать с верандой/пристройкой
 • Если веранда на всю длину, ширина становится больше везде. Тогда в каждом ряду берём ширину W + вынос веранды → больше свай в ряду.
 • Если веранда на части длины (например, только на половине дома):
 • Ряды там, где веранды нет → считаем по основной ширине W.
 • Ряды там, где веранда есть → считаем по увеличенной ширине (W + вынос веранды).
 • Итог = сложить сваи по всем рядам: где-то 3 в ряд, где-то 5 — сложили и получили общий счёт.

⸻

Шаг 5. Считаем
 • Без веранды:
Итог = (ряды по длине) × (сваи в ряду по ширине).
 • С верандой не на всю длину:
Складываем по рядам:
«ряд без веранды» + «ряд на стыке» + «ряд с верандой» … = всего свай.

⸻

Быстрая шпаргалка (самые частые размеры)
 • 2×2 / 2×3 → 2 ряда × 2 в ряд = 4 сваи
 • 2×4 / 2×5 / 2×6 → 3 ряда × 2 = 6 свай
 • 3×3 → 2 ряда × 3 = 6 свай
 • 3×4 / 3×5 / 3×6 → 3 ряда × 3 = 9 свай
 • 6×3 → 3 ряда × 3 = 9 свай
 • 6×4 → 3 ряда × 4 = 12 свай
 • 7×3 → 4 ряда × 3 = 12 свай
 • 7×4 → 4 ряда × 4 = 16 свай
 • 6×7 → 3 ряда × 7 = 21 свая

⸻

Примеры с верандой (чтобы не запутаться)

Пример А: 6×3 + веранда 1,5 м на всю длину
 • Рядов по длине: 3 (0–3–6 м)
 • Ширина стала 3 + 1,5 = 4,5 → в каждом ряду 5 свай
 • Итого: 3×5 = 15 свай

Пример Б: 6×3 + веранда 1,5 м только на половине длины (с 3 до 6 м)
 • Рядов: 3 (0–3–6 м)
 • На 0 м веранда не попадает → ширина 3 м → 3 сваи
 • На 3 м и 6 м веранда есть → ширина 4,5 м → 5 свай и 5 свай
 • Итого: 3 + 5 + 5 = 13 свай

⸻

Частые ошибки (и как не ошибаться)
 1. Считать только по площади.
Нельзя. Площадь нужна для доставки, а сваи — по рядам и ширине.
 2. Забыли, что веранда не на всю длину.
Тогда получается лишние сваи. Смотрим: где веранда есть → шире; где нет → как было.
 3. Сделали слишком большой промежуток между рядами по длине.
Держим ≤ 3 м.
 4. Поставили 1 сваю в ряду.
Так нельзя. Минимум 2 — всегда.

⸻

Если очень торопимся (супер-короткая версия)
 1. Найди ряды по длине: через ~3 м, края всегда.
 2. В каждом ряду поставь минимум 2 сваи и дальше — по метру ширины.
 3. Веранда увеличивает ширину только там, где она есть.
 4. Сложи сваи по всем рядам — это и есть итог.
